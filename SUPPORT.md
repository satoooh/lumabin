# Support

LumaBin は個人開発のOSSです。商用サポートやSLAはありませんが、再現性のある報告と小さな改善提案を歓迎します。

## サポート対象

- 最新の GitHub Release
- `main` branch の現在の実装
- macOS 上の Electron desktop app
- Cloudflare R2 と generic S3 の core operation

## サポート対象外

- private bucket や実ユーザーデータを含む調査
- access key、secret key、署名済みURL、Apple署名資格情報を共有する必要がある調査
- provider固有のACL/policy/bucket管理UI
- 古いReleaseやfork独自変更の継続サポート

## 質問・不具合報告

- バグ報告: [Bug report](https://github.com/satoooh/lumabin/issues/new?template=bug_report.yml)
- 機能提案: [Feature request](https://github.com/satoooh/lumabin/issues/new?template=feature_request.yml)
- セキュリティ問題: [SECURITY.md](SECURITY.md)

報告には、機密情報を伏せた再現手順、期待結果、実際の結果、実行した検証を含めてください。
