# Code of Conduct

LumaBin は、安心して参加できるOSSプロジェクトであることを重視します。

## 期待するふるまい

- 相手の時間、背景、経験値を尊重する
- 技術的な指摘は、人格ではなく具体的なコード、設計、再現条件に向ける
- 不確かな内容は推測として明示する
- セキュリティやプライバシーに関わる情報を公開issueやPRに貼らない
- 意見が分かれる場合は、要件、テスト、実装上の制約に戻って議論する

## 許容しないふるまい

- ハラスメント、差別、脅迫、侮辱
- 個人情報、資格情報、private bucket 名、private object key、署名済みURLの公開
- 悪用手順や攻撃コードの公開
- 議論を妨げるスパム、荒らし、繰り返しの脱線

## 適用範囲

この行動規範は、GitHub issue、pull request、discussion、release note、その他このプロジェクトに関わる公開コミュニケーションに適用されます。

## 対応

maintainer は、必要に応じてコメントの編集依頼、非表示、issue/PRのロック、参加制限を行います。セキュリティ問題は [SECURITY.md](SECURITY.md) に従って非公開で報告してください。
